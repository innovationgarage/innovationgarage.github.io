How to use our code

We publish many of our components and libraries as Open Source / Free Software on GitHub. That means that you can use them in your own projects (under certain conditions described in the license file accompanying the respective project)!

All components and libraries come with a README that at the very least describes how to compile / install the software and how to make a minimal use of it. In some cases a full reference manual is provided. Regardless, the software is provided as-is and there are no warranties as to functionality.

To start using our code, simply clone the git repo, check the LICENSE file to make sure the terms are compatible with your intended use and then follow the instructions in the README file.

You are welcome to report any bugs in the github issue tracker for each component or library, and we will do our best to look into the issue. However, note that this software is not included in any of our support contracts.
