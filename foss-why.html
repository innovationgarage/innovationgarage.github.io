Why Open Source?

We publish many of our components and libraries as Open Source / Free Software on GitHub. That means that you can use them in your own projects under certain conditions described in the license file accompanying the respective project. Why do we do this?

Flexibility - by publishing the code for you to use in your projects we can support many more use cases than by simply providing a single commercial product. Together we can build business opportunities around the code neither of us would have had, had we not shared the code.

Auditability - by publishing the code you can inspect our code to ensure that it does what we said it would do - in great detail.

Quality improvement - by letting you not only report problems in our software, but inspect the software for the source of the errors as well as potentially fix them yourself we can greatly improve the quality of the software.
